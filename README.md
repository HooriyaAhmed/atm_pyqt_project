ATM MACHINE PROJECT
Project Overview:

This project is a console-based Mini Banking System developed in Python that simulates real-world banking operations such as user registration, login authentication, deposits, withdrawals, and money transfers between accounts. It also maintains a secure transaction history and includes an account lock system for failed login attempts.

Key Features:
1. User Authentication System
New users can register with a secure PIN
Unique account number is generated automatically
Login system validates account number and PIN
After 3 wrong attempts, account gets locked automatically
