## 🚀 💡 Project Overview
✨ A **console-based Mini Banking System** built in Python that simulates real-world banking operations like:
- User registration 🔐  
- Login authentication 🔑  
- Deposits & Withdrawals 💰  
- Money transfers 🔄  
- Transaction history 📊  

---

## 🎯 ✨ Features

### 🔐 Authentication System
- Register new users with PIN  
- Auto-generated account numbers  
- Secure login system  
- 🔥 Account lock after 3 failed attempts  

---

### 💰 Banking Operations
- Deposit money instantly  
- Withdraw with balance check  
- Real-time balance updates  

---

### 🔄 Transfer System
- Send money between accounts  
- Dual transaction logging  
- Safe balance validation  

---

### 📊 Mini Statement
- View transaction history  
- Track all account activity
